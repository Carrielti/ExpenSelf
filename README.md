# 💰 ExpenSelf

Sistema web de controle financeiro pessoal desenvolvido para auxiliar usuários no gerenciamento de despesas, receitas e visualização de dados financeiros.

---

## 🌐 Projeto Online

Acesse o projeto:

https://www.ecomap.wuaze.com

---

## 📌 Sobre o Projeto

O ExpenSelf foi desenvolvido com o objetivo de facilitar o controle financeiro pessoal, permitindo o registro de despesas, organização por categorias e acompanhamento dos gastos por meio de gráficos e relatórios.

O projeto foi criado como uma aplicação web utilizando front-end, back-end e banco de dados, sendo uma das minhas principais experiências práticas com desenvolvimento full stack.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Python
- Firebase
- InfinityFree
- Render

---

## ✨ Funcionalidades

- Cadastro de usuários
- Login de usuários
- Registro de despesas
- Listagem de despesas
- Edição de despesas
- Exclusão de despesas
- Categorização de gastos
- Geração de gráficos
- Exportação de relatório
- Integração com banco de dados
- Integração com Firebase
- API Python hospedada externamente

---

## 📊 Objetivo do Sistema

O sistema busca ajudar o usuário a visualizar melhor seus gastos, identificar categorias com maior impacto financeiro e organizar sua vida financeira de forma simples e acessível.
